Create User

Creates user in Sheets via POST /manager/users/create. Your Worker hashes passwords in /users/reset-password, so this UI uses:
1) create user with placeholder hash
2) optionally reset to a temp password
Note: “Create + set temp password” will force mustChangePassword=true.

Reset Password

Manager can reset anyone by userId or username via POST /users/reset-password.
After reset, user will be forced to change password on next login (mustChangePassword=true).
